# Robot Soccer — Digital Twin

A digital-twin robot soccer project combining a software simulation and physical omniwheel robots playing against each other, built on the open-source [Robot Soccer Kit](https://robot-soccer-kit.github.io/) platform.

## Overview

The setup pairs a **software simulation** with **physical robots** (provided by the school) so that strategies can be developed and tested digitally, then run identically on real hardware — a digital-twin approach to robot soccer.

The Robot Soccer Kit platform provides:

- Omnidirectional-wheel robots with Bluetooth communication
- A camera-based external tracking system (marker tags) to locate robots and the ball on the field
- A Python API to control the robots and read the game state

## My Role

## Tech Stack

- **Language:** Python (via the Robot Soccer Kit API)
- **Hardware:** Omniwheel robots (Robot Soccer Kit), Bluetooth communication
- **Vision:** Camera-based marker tracking system

## Key Technical Points

- Writing game strategy/control code that behaves consistently in both simulation and on physical robots
- Working within a fixed hardware/API platform (Robot Soccer Kit) rather than building the robot from scratch
- Coordinating multiple robots in real time based on live position tracking
